# MiraKuru MVP

MiraKuru is a value-validation MVP for helping young people turn everyday screen time, games, videos, smartphone use, and future anxiety into short self-understanding records and small next actions.

This MVP is intentionally simple:

- no login
- runs as a static HTML file
- stores records only in the browser on the current device
- includes a feedback flow for early validation

## Files

- `mvp.html`: public MVP experience
- `mirakuru-response-logic.json`: response logic reference
- `mirakuru-blp-experience-mvp-spec.md`: MVP specification for BLP review
- `mirakuru-product-core.md`: product positioning and core concept

## Local Use

Open `mvp.html` in a browser.

## Safety Note

MiraKuru is not a medical or emergency support service. If the input suggests immediate danger or self-harm, the MVP switches to a safety-first message and asks the user to contact nearby adults, school staff, local support, or emergency services.
