# MiraKuru Product Core

## 1. MiraKuruとは何か

MiraKuruは、誰もが持つ「才能の原形」を日々の体験から見つけ、点と点をつなぎ、AI時代に通用するユニークネスへ育てるAIアプリ。

スクリーンの中や日常で過ごした時間を、責めずに受け止め、未来につながる小さな一歩へ変えていく。

YouTube、ゲーム、SNS、趣味、何もできなかった日、不安な日。  
外から見ると「ただ見ていた」「ただ遊んでいた」「止まっていた」に見える時間にも、本人だけの興味、安心、憧れ、違和感、不満、観察、制作の種がある。

MiraKuruは、その一次体験をAIとの対話で言語化し、好き、不安、違和感、小さな行動の中にある才能の原形を見つけ、進路や制作、起業の材料へ変えていく。

一言で言うと、

**誰もが持つ「才能の原形」を日々の体験から見つけ、点と点をつなぎ、AI時代に通用するユニークネスへ育てるAIアプリ。**

## 1.1 最終ゴール

MiraKuruの最終ゴールは、若者が**自分も相手も大事にしながら、価値を作れるようになること**。

そのために、次の道筋をたどる。

```text
自分を知る
↓
強み・武器を見つける
↓
誰かの困りごとに気づく
↓
小さく作る・動く
↓
自分も相手も大事にしながら価値を作る
```

MiraKuruは、自己理解だけで終わらない。  
起業アイデアだけを出すわけでもない。

自分の好き、不安、違和感、スクリーン時間、一次体験を、自分の強みとして見つけ直し、誰かの困りごとに向けて小さく使えるようにする。

## 2. MiraKuruが扱う痛み

### 課題の一文

若者は、日々スクリーンの中で多くの興味や不安に触れているが、それを自分のスキルや進路、制作の一歩に変える方法を持っていない。

MiraKuruが扱うのは、若者が日常の中で感じる、まだ言葉になりきっていない痛み。

- 好きなことが将来にどうつながるかわからない
- YouTube、ゲーム、SNSに時間を使ったあと罪悪感がある
- 何か作りたいが、何を作ればいいかわからない
- 起業したいが、アイデアが出ない
- 将来が怖い
- 周りと比べて落ち込む
- 親や先生に責められて、自分の時間が無価値に見える
- 何もできなかった日をどう扱えばいいかわからない
- 自分にはスキルや実績がないと思っている
- 自分の体験を言語化できない

## 3. 対象ユーザー

### 中心ユーザー

スマホ、ゲーム、SNS、動画に日々触れながら、自分の好きや不安を将来につなげる方法がわからない10代・若者。

### 特に深く刺さる人

- 将来不安がある人
- 進路に迷っている人
- 不登校経験がある人
- 通信制高校やフリースクールなど、多様な学びの場にいる人
- 起業や制作に興味はあるが、最初の一歩がわからない人
- 親や先生から「何をしているの」と言われてしんどい人
- スマホやゲームの時間に罪悪感がある人

### 将来的な支払者・利用者

- 本人
- 保護者
- 学校
- 通信制高校
- フリースクール
- 探究学習・キャリア教育の支援者
- 若者支援施設

## 4. MiraKuruではないもの

MiraKuruは、以下ではない。

- スマホ時間を管理・制限するアプリ
- ゲームやYouTubeをやめさせるアプリ
- 「頑張れ」と励ますアプリ
- 「大丈夫」と安心させるアプリ
- メンタルヘルス治療アプリ
- 学校復帰を目的にしたアプリ
- 就職・転職だけを目的にしたキャリア診断
- 起業で成功する方法を教えるアプリ
- ChatGPTの薄いラッパー
- 毎日完璧に続けることを求める習慣アプリ

MiraKuruは、若者に「ちゃんとしろ」と言わない。  
今いる場所から、未来へつながる小さな線を一緒に探す。

## 5. コア思想

### 減らすんじゃない。つなげる。

スクリーン時間を最初から悪として扱わない。

YouTubeを見ていた時間、ゲームをしていた時間、SNSを眺めていた時間は、浪費だけではなく、安心、退屈の回避、興味、観察、憧れ、社会との接点だった可能性がある。

MiraKuruは、まずその時間が何を果たしていたのかを見る。

### 励ますんじゃない。隣にいる。

「頑張れ」「大丈夫」「なんとかなる」は、苦しい人には重すぎることがある。

MiraKuruは、無責任に明るくしない。  
前を向けない日にも隣にいることから始める。

### 答えを出すんじゃない。問いを育てる。

AIが正解を出すのではなく、ユーザーの一次体験から問いを育てる。

- 何に反応したのか
- どこが好きだったのか
- 不満はあるのか
- 不満がないなら、もっと良くできるところはあるのか
- 自分なら少しだけ何を変えるか
- 小さく作るなら何ができるか

### 実績を押しつけない。小さい証拠を残す。

MiraKuruは、若者に「実績を作れ」と急かさない。

ただ、責められた日や不安な日にも、自分の側に小さな証拠を残すことはできる。

- メモ
- ラフ
- 写真
- 好きな理由3つ
- 使ったツール
- 小さな制作物
- 誰かに聞いた反応

これらを積み上げて、スキルや未来の材料にする。

## 6. コア体験

MiraKuruの体験は、次の流れでできている。

1. **記録する**
   - 今日見たもの、遊んだもの、気になったこと、不安だったことを短く書く。

2. **受け止める**
   - AIが責めずに受け止める。
   - いきなり改善しない。

3. **少しだけ聞く**
   - 1つだけ深掘りする。
   - 答えやすい選択肢を出す。

4. **構造化する**
   - 好き、違和感、不満、改善余地、隠れたスキル、興味を抽出する。

5. **小さく作る**
   - 10分でできる制作案、メモ案、観察案を出す。

6. **証拠にする**
   - 作ったもの、考えたこと、反応したものを記録として残す。

7. **未来の地図にする**
   - 記録を積み上げ、興味、スキル、制作物、進路候補、起業アイデアとして見える化する。

## 7. 会話原則

### 言っていい方向

- 今日は記録だけで終わってもいい
- 怖いものは、怖いままでいい
- 無理にいい感じにまとめなくていい
- その時間が何から守ってくれていたのか見てみよう
- 不満がないなら、もっと良くできるところを探してみよう
- 作れないなら、好きなところを3つ書くだけでもいい
- 売る前に、まず自分のオリジナルとして小さく作ってみよう
- これは解決じゃない。でも記録にはなる

### 避ける言葉

- 頑張ろう
- 大丈夫
- なんとかなる
- 君ならできる
- 未来は明るい
- わかってると思うけど
- それはわかってるよね
- 比べなくていい
- 人は人、自分は自分
- 明日は減らそう
- スマホ依存だね
- まず行動しよう
- 起業家なら行動あるのみ
- 絶対売れる
- 実績を作れば認められる

### 不安が強い時

未来につなげない。

まず、痛みを増やさない。  
深掘りしすぎない。  
記録だけで終わる選択肢を残す。

### 危険なサインがある時

自傷や希死念慮など、AIだけで抱えるには重い入力がある場合は、制作提案や深掘りを止める。

身近な人、専門窓口、緊急窓口につながるよう促す。  
MiraKuruは医療・心理支援の代替を名乗らない。

## 8. 独自価値

### 独自価値の一文

MiraKuruの独自価値は、スマホやゲームの時間を制限するのではなく、好きや不安を言語化し、未来への小さな一歩に変えること。

MiraKuruの独自価値は、AIの性能そのものではなく、**若者が自分の体験を未来につなげるための体験設計**にある。

| 既存のもの | MiraKuru |
| --- | --- |
| スクリーン時間を制限する | スクリーン時間を未来の材料にする |
| 日記を書く | 体験をスキル・制作案に変える |
| ChatGPTに自分で聞く | MiraKuruが問いを届ける |
| 励ます | 無責任に明るくしない |
| キャリア診断をする | 一次体験から進路の材料を作る |
| 起業アイデアを出す | 好き・不満・身近な困りごとから小さく検証する |

## 9. 既存AIとの違い

ChatGPTでも、単発では似たことができる。

しかし、MiraKuruは「答えるAI」ではなく、若者の好きや不安を、余力に合わせて言語化し、強みと小さな一歩に変える専用フローである。

### 一言での違い

**ChatGPTは、聞かれたことに答える。MiraKuruは、聞けない日にも、好きや不安を強みと小さな一歩に変える。**

具体的には、MiraKuruは次が違う。

- ユーザーが毎回プロンプトを考えなくていい
- 苦しい時にも答えやすい問いが届く
- 会話がMiraKuruの安全設計に従う
- 記録が蓄積される
- 好き、スキル、制作案、未来候補として構造化される
- 週次・月次で自分の変化が見える
- 作れなかった日、戻ってきた日も扱える

一言で言うと、

**ChatGPTは道具。MiraKuruは、一次体験を未来に変える習慣と資産化の仕組み。**

## 10. 初期MVPの境界線

### MVPの一文

最初のMVPは、今日見たもの・感じたことを入力すると、AIが責めずに対話し、好き・不安・隠れたスキル・10分でできる小さな一歩を返すWebアプリ。

### 最初に作る

- 体験入力
- AIとの短い対話
- 会話モード判定
  - YouTubeを見た日
  - ゲームをした日
  - 何もできなかった日
  - 将来が怖い日
  - 起業アイデアを探したい日
  - 親や先生に責められた日
  - スマホを見すぎて罪悪感がある日
  - 人と比べて落ち込んだ日
- 好き、不満、改善余地、隠れた興味、スキル、制作案の抽出
- 記録一覧
- 週次レポート
- 簡易スキル・興味マップ

### 最初は作らない

- 完全なスクリーンタイム自動連携
- ネイティブアプリ
- 人間メンターとのマッチング
- 学校向け管理画面
- SNS機能
- 大規模なポートフォリオ生成
- 収益化マーケットプレイス

## 11. 将来拡張

### Screen Insight

スクリーンタイムや手入力のアプリ利用情報をもとに、どのアプリのどんな要素に反応しているのかを見る。

目的は制限ではなく、好きの構造化。

### Portfolio

日々の記録、制作物、学び、スキルをポートフォリオ化する。

進路、探究学習、自己紹介、起業アイデアに使える形へ変換する。

### School / Support

本人の同意を前提に、学校、フリースクール、支援者が若者の興味や小さな変化を理解できるようにする。

本人の尊厳を守る。監視ツールにしない。

### Human Mentor

将来的に、実際に働く人、作っている人、起業している人へ質問できる導線を作る。

最初から通話マッチングにせず、AI職業インタビュー、記事推薦、非同期Q&Aから始める。

## 12. 収益化の基本方針

課金対象は、記録そのものではなく、**体験の変換価値**。

- 日記をスキルに変える
- スクリーン時間を興味マップに変える
- 好きを制作案に変える
- 小さな行動をポートフォリオに変える
- 進路や起業の材料に変える

初期案:

- Free: 回復しない無料クレジット3回
- Plus: 月40回のAI対話、記録保存、週次レポート、スキル・興味マップ
- Pro: 月100回のAI対話、ポートフォリオ、起業アイデア整理、外部連携
- School / Support: 学校・支援施設向け

プランの役割:

- Free: 体験する
- Plus: 自分を知る
- Pro: 強みを活かす

## 13. 守るべき倫理

- ユーザーを責めない
- スマホ時間を一方的に悪としない
- 頑張れ、大丈夫を安易に言わない
- 理解したふりをしない
- ユーザーの不安を軽く扱わない
- 既存作品の権利を侵害する制作や販売へ誘導しない
- 医療・心理支援の代替を名乗らない
- 未成年のデータ保護を最優先する
- 保護者や学校向け機能では、本人の尊厳と同意を重視する
- 監視や管理ではなく、本人の未来接続を目的にする

## 15. いまの仮説

若者にとって、スクリーン時間は単なる浪費ではなく、居場所であり、興味の入口でもある。

一方で、多くの若者は、その体験を将来やスキルに変える方法を持っていない。

MiraKuruは、スクリーン時間を制限するのではなく、好き、違和感、改善余地、小さな制作案へ変換し、未来につなげる。
