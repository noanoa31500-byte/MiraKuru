## 1. 3分プレゼンで必ず答えること

### ① ビジネスプラン、主要顧客、課題

**ビジネスプラン**

MiraKuru は、若者が好きや不安を言葉にして、未来への小さな一歩を踏み出すためのAIアプリ。

YouTube、ゲーム、SNSなどのスクリーン時間や日常の体験を、AIとの対話で「好き」「違和感」「小さな制作案」「スキルの証拠」に変えていく。

**主要顧客**

最初の主要顧客は、スマホ、ゲーム、SNS、動画に日々触れながら、自分の好きや不安を将来につなげる方法がわからない10代・若者。

その中には、不登校経験や進路不安を抱える人も含まれる。将来的な顧客は、学校、通信制高校、フリースクール、探究学習・キャリア教育の支援者、子どものスマホ時間を責めずに未来へつなげたい保護者。

**課題**

若者は、日々スクリーンの中で多くの興味や不安に触れているが、それを自分のスキルや進路、制作の一歩に変える方法を持っていない。

スマホやゲームやSNSに時間を使っている体験は、周囲からは「無駄」「遊び」と見られやすいが、本人の中には、興味、安心、憧れ、不満、観察、制作の種がある。

問題は、その体験が言語化されず、スキルや進路、制作物につながらないこと。

### ② 独自価値

MiraKuru の独自価値は、**スマホやゲームの時間を制限するのではなく、好きや不安を言語化し、未来への小さな一歩に変えること**。

既存のスクリーンタイム系アプリは、制限、ブロック、管理に寄りやすい。汎用AIは相談すれば答えてくれるが、苦しい人ほど何を聞けばいいかわからないし、毎日続ける設計も弱い。

MiraKuru は、若者に合わせた問い、励ましすぎない言葉、体験の蓄積、スキル地図化、小さな制作案までを一体化する。

一言で言うと、**「見ていた時間」を「作れる力」に変えるアプリ**。

### ③ キーサクセスファクター

最も重要な要素は、**若者が「ここなら書ける」と感じる会話設計と安全設計**。

理由は、このサービスの価値がAIの性能そのものではなく、好きや不安を否定されずに言葉にできる体験にあるから。

たとえば、将来が怖い人に「大丈夫」と言わない。何もできなかった人に「明日は頑張ろう」と言わない。スマホを見すぎた人に「減らそう」と言わない。まず、その時間が何から守ってくれていたのかを見る。

この言葉づかいが信頼を作り、継続を生み、日々の体験を価値あるデータに変える。

### ④ 壁と乗り越え方

想定される壁は4つ。

1. **AIの返答が無責任な励ましや説教になること**
   - 乗り越え方: NG表現、会話プロトタイプ、安全ルールを先に設計し、危険サインでは専門窓口や身近な人につなげる。

2. **ChatGPTでもできると言われること**
   - 乗り越え方: MiraKuruは「答えるAI」ではなく、若者の好きや不安を、余力に合わせて言語化し、強みと小さな一歩に変える専用フローで差別化する。

3. **本当に若者が使い続けるか**
   - 乗り越え方: まずはWebまたはLINE風プロトタイプで5〜10人に試し、「責められない感じがあるか」「未来につながる感じがあるか」を検証する。

4. **精神的にしんどい人を扱う責任**
   - 乗り越え方: 医療・心理支援の代替を名乗らず、深刻な入力には人間の支援につなげる。学校・保護者向け機能では本人の尊厳と同意を重視する。

## 3. スライド構成

### Slide 1. タイトル

**MiraKuru**  
スクリーンの時間を、未来の種に。

話すこと:

- 代表者名
- サービス名
- 1行コンセプト

### Slide 2. 自己紹介・原体験

**「頑張れ」「大丈夫」がつらい時がある**

話すこと:

- 不登校の頃、励ましの言葉が重かった
- でもこれは不登校だけではなく、将来不安を抱える若者全体にある痛み
- スマホやゲームの時間を責めずに、未来へつなげたい

### Slide 3. 現状課題

**スクリーン時間は、浪費として見られやすい**

入れる要素:

- 若者は、自分の好きや日々の体験を価値に変えにくい
- YouTube、ゲーム、SNSに時間を使うほど、「このままでいいのか」という不安が残る
- 汎用AIは「何を聞くか」が必要

参考データ:

- 文部科学省の令和6年度調査では、小中学校の不登校児童生徒数は約35万4千人、高校は約6万8千人。
- こども家庭庁の令和7年度速報では、インターネット利用者の平均利用時間は高校生で約6時間44分、中学生で約5時間24分。

### Slide 4. 顧客

**主要顧客: 好きと将来がつながらない10代・若者**

具体像:

- 中高生、通信制高校生、大学生、若手
- YouTube、ゲーム、SNSに長く時間を使う
- 「このままでいいのか」と不安がある
- 自分の好きや日常が将来にどうつながるかわからない
- 起業、制作、進路に興味はあるが、最初の一歩が見えない

支払者候補:

- 本人
- 保護者
- 学校、フリースクール、支援施設

### Slide 5. 独自価値

**減らすんじゃない。つなげる。**

比較:

| 既存サービス | MiraKuru |
| --- | --- |
| スクリーン時間を制限する | スクリーン時間を未来の材料にする |
| 日記を書く | 体験をスキル・制作案に変える |
| ChatGPTに自分で聞く | MiraKuruが問いを届ける |
| 励ます | 無責任に明るくしない |

このスライドで補足する論理:

- 若者は、不安が強いほど「何を聞けばいいか」がわからなくなる
- その状態では、正論や励ましよりも、まず防衛をほどく会話が必要
- MiraKuruは、責めない問いから始めることで、自己否定を言語化へ変え、小さな実行に接続する

入れると強い一文:

> MiraKuruは、答えを押しつけるAIではなく、
> 自己否定の強い若者が「ここなら書ける」と感じる会話設計を持つ。

### Slide 6. サービス概要

**記録 → 対話 → 構造化 → 小さく作る → 未来の地図**

機能:

- 今日見たもの、遊んだものを1つ記録
- AIが責めずに深掘り
- 好き、不満、もっと良くできる点を抽出
- 10分でできる制作案を提案
- スキル・興味マップに蓄積

このスライドで明示すること:

- しんどい日は無理に強み化しない
- まず「何に反応したか」「その時間が何から守っていたか」を見る
- 余力がある日にだけ、才能の原形と小さな一歩へ進む

入れると強い一文:

> 未来が見えない時に必要なのは、正しい答えではなく、
> 今の体験に意味があるかもしれないと感じられる変換体験である。

