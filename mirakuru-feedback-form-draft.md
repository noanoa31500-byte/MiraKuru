# MiraKuru Feedback Form Draft

## Form Title

MiraKuru についてのフィードバック

## Form Description

MiraKuruは、スマホ・ゲーム・動画・SNS・将来不安など、若者の日常の出来事を責めずに整理し、未来につながる小さな一歩へ変えるAIアプリです。

外から見ると「ただ見ていた」「ただ遊んでいた」「何もしていない」に見える時間の中にも、本人だけの好き、違和感、憧れ、不満、観察、問い、制作の種があるかもしれません。MiraKuruは、その時間をすぐに否定せず、短い入力や選択肢から、興味のヒント、強みの手がかり、次にできそうなことへつなげます。

たとえば、ゲーム実況を見ていた日なら、ただ「ゲームを見た」で終わらせず、話し方、世界観、攻略、仲間感、編集のテンポなど、どこに反応したのかを整理します。スマホを見すぎて罪悪感がある日なら、いきなり「明日は減らそう」とは言わず、その時間が何から少し守ってくれていたのか、何に引っかかったのかを見ます。将来が不安な日なら、無責任に「大丈夫」と言わず、不安を小さく分けて扱います。

製品版では、日々の記録やチャット履歴から、何度も出てくる興味・不安・反応の形を見つけ、MiraKuru Graphとして見えるようにします。Graphでは、自分でも気づきにくい強みの組み合わせ、向いていそうな制作・探究テーマ、進路や小さな事業アイデアにつながる材料を見つけていきます。

MiraKuruは、スマホ時間を管理するアプリでも、ただの日記アプリでも、ChatGPTの薄いラッパーでもありません。目指しているのは、若者が自分も相手も大事にしながら、自分の体験を価値に変えていけるようにすることです。

このフォームでは、MiraKuruの考え方が伝わるか、どの機能に価値を感じるか、普通のAIチャットや日記アプリと違うと感じるかを確認します。

### Q3. MiraKuruの考え方で、いちばん価値を感じたものはどれですか？

- スマホやゲームの時間を、すぐ悪いものとして扱わないこと
- 「頑張れ」「大丈夫」と無責任に励まさないこと
- 長く説明できない日でも、選ぶだけ・一言だけで始められること
- 好きや不安を、強みや制作のヒントに変えること
- 記録が増えるほど、自分の反応のパターンが見えること
- 特に価値を感じなかった

### Q4. 製品版で一番使ってみたい機能はどれですか？

- 日々の記録から、自分の興味や不安のパターンが見える

- 好きなことを、制作や探究テーマにつなげられる
- 将来不安を、進路や小さな行動に分けられる
- 起業や小さな事業アイデアにつなげられる
- 学校や保護者に、自分の状態や可能性を説明しやすくなる
