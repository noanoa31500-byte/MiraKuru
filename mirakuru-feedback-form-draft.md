# MiraKuru MVP Feedback Form Draft

## Form Title

MiraKuru 体験MVP フィードバック

## Description

MiraKuruは、スマホ・ゲーム・動画・SNS・将来不安など、日常の出来事を責めずに整理し、次にできそうなことへつなげるAIアプリのMVPです。

このフォームでは、入力しやすさ、責められない感じ、普通のAIチャットや日記アプリとの違い、また使いたいかを確認します。

## Questions

1. あなたに近いものはどれですか？
   - 本人
   - 保護者
   - 教育・支援関係者
   - その他

2. 最初に選んだ入口はどれですか？
   - ゲーム・動画のこと
   - スマホ時間が少し気になる
   - 最近、将来が不安
   - まだ言葉になっていない

3. 入力しやすかったですか？
   - はい
   - 少し
   - いいえ

4. 責められない感じはありましたか？

   - 少し
   - いいえ

5. 使ったあと、不安や罪悪感に変化はありましたか？
   - 軽くなった
   - 少し軽くなった
   - 変わらない
   - 重くなった
   - もともと不安や罪悪感はなかった

6. 今日のまとめで、自分の反応が少し見えましたか？
   - はい
   - 少し
   - いいえ

7. 提示された「次にできそうなこと」は重すぎませんでしたか？
   - ちょうどいい
   - 少し重い

8. 普通のAIチャットや日記アプリと違う感じはありましたか？
