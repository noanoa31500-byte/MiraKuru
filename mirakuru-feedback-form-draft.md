# MiraKuru MVP Feedback Form Draft

## Form Title

MiraKuru 体験MVP フィードバック

## Description

MiraKuruは、スマホ・ゲーム・動画・SNS・将来不安など、若者の日常の出来事を責めずに整理し、興味のヒントや次にできそうなことへつなげるAIアプリのMVPです。

今回のMVPは完成版ではありません。まずは、短い入力でも始められるか、責められない感じがあるか、普通のAIチャットや日記アプリとは違う価値を感じるかを確認するための検証版です。

製品版では、チャット履歴や日々の記録から、何度も出てくる興味・不安・反応の形を見つけ、自分でも気づきにくい強みの組み合わせを見えるようにすることを目指しています。

このフォームでは、MiraKuruを実際に使ったあとに、入力のしやすさ、表示された言葉の受け取りやすさ、気持ちの変化、次にできそうなことの重さ、また使いたいかを確認します。良かった点だけでなく、違和感があった点も重要な検証材料になります。

回答に正解はありません。感じたことに近いものを選んでください。

## Questions

1. あなたに近いものはどれですか？
   - 説明: どの立場の人が使ったときに価値を感じるかを確認するための質問です。
   - 本人
   - 保護者
   - 教育・支援関係者
   - その他

2. 最初に選んだ入口はどれですか？
   - 説明: どの入口が使われやすいか、また入口ごとに返答の受け取り方が違うかを確認します。
   - ゲーム・動画のこと
   - スマホ時間が少し気になる
   - 最近、将来が不安
   - まだ言葉になっていない

3. 入力しやすかったですか？
   - 説明: 長く説明しなくても始められる設計になっているかを確認します。
   - はい
   - 少し
   - いいえ

4. 責められない感じはありましたか？
   - 説明: MiraKuruが大切にしている「スマホやゲームの時間をすぐ悪いものにしない」体験が伝わったかを確認します。
   - はい
   - 少し
   - いいえ

5. 使ったあと、不安や罪悪感に変化はありましたか？
   - 説明: 不安や罪悪感を無理に消すことではなく、少し整理された感覚があるかを確認します。
   - 軽くなった
   - 少し軽くなった
   - 変わらない
   - 重くなった
   - もともと不安や罪悪感はなかった

6. 今日のまとめで、自分の反応が少し見えましたか？
   - 説明: 日常の出来事から、興味や反応のヒントが見える体験になっていたかを確認します。
   - はい
   - 少し
   - いいえ

7. 提示された「次にできそうなこと」は重すぎませんでしたか？
   - 説明: MiraKuruの提案が、無理な行動ではなく、実際に試せそうな小ささになっているかを確認します。
   - ちょうどいい
   - 少し重い
   - かなり重い
   - もっと具体的にしてほしい

8. 普通のAIチャットや日記アプリと違う感じはありましたか？
   - 説明: MiraKuruの独自価値が伝わっているかを確認するための重要な質問です。
   - はい
   - 少し
   - いいえ

9. MiraKuru Graphが育つなら、また使いたいですか？
   - 説明: 記録が増えることで、自分でも気づきにくい興味・不安・反応の形が見える機能に価値を感じるかを確認します。
   - はい
   - 少し
   - いいえ

10. 一番よかった点、または引っかかった点を教えてください。
    - 説明: 選択肢では拾えない感想を知るための質問です。短くても大丈夫です。
    - 自由記述

11. 追加ヒアリングに協力してもよい場合、連絡先を書いてください。
    - 説明: さらに詳しく話を聞いてもよい人だけ任意で書いてください。無理に書く必要はありません。
    - 自由記述
    - 任意
